# Customer Churn Prediction Dashboard

This project is an end-to-end Machine Learning solution designed to predict customer churn and provide actionable business insights for customer retention strategies. Using a telecom customer dataset, the project performs data preprocessing, exploratory data analysis (EDA), feature engineering, model training, evaluation, and deployment through a Streamlit web application.

The project also includes a Power BI dashboard for business stakeholders to monitor churn trends, customer behavior, high-risk customer segments, and revenue impact. By combining predictive analytics with interactive visualizations, the solution helps organizations identify customers likely to leave and take proactive retention measures.

## Key Features

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Customer Churn Prediction using Random Forest
* Model Evaluation (Accuracy, Precision, Recall, F1-Score, ROC-AUC)
* Hyperparameter Tuning
* Streamlit Web Application
* Power BI Interactive Dashboard
* High-Risk Customer Identification
* Churn Probability and Risk Segmentation
* Deployment-Ready Architecture

## Tech Stack

* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn
* Streamlit
* Power BI

## Project Outcome

The model predicts whether a customer is likely to churn and provides churn probabilities that can be used for customer retention campaigns. The Power BI dashboard offers executive-level insights into churn trends, customer segments, and business performance metrics.
